# Java

In the ealy 1990's, this high-level, general-purpose language was founded and designed by [James Gosling](https://www.computerhistory.org/fellowawards/hall/James-Gosling/).

- Java source cocde is compiled into [bytecode](https://techterms.com/definition/bytecode), which can be processed by the Java Virtual Machine (JVM).

- Java is being developed and maintained by [Oracle](https://www.oracle.com/index.html).

- Java's syntax is very similar to that of [C++](../cplusplus/description.md)
